Descrição dos Valores de Display

  • block: Elemento ocupa toda a largura disponível. Ex: <div>, <p>, <form>.
  • inline: Elemento flui na mesma linha. Ex: <span>, <a>.
  • inline-block: Como inline, mas permite definir largura e altura. Útil para layouts lado a lado.
  • flex: Cria um container flexível para alinhar itens com controle de direção, espaçamento e alinhamento.
